## 📖 Sobre o Jogo

**CHARADAS - O Jogo de Tabuleiro** é uma experiência nostálgica que combina o raciocínio das charadas com a diversão dos jogos de tabuleiro tradicionais. Conquiste territórios, ganhe estrelas e torne-se uma lenda!

### 🎯 Inspiração
- **Grow**: Banco Imobiliário, War, Detetive
- **Estrela**: Resta Um, Lig-4, Damas

## 🎮 Como Jogar

### Regras Básicas
1. Abra o jogo e veja os 9 territórios (cards)
2. Leia a charada de cada um
3. Tente adivinhar a resposta
4. **Clique no card** para revelar a resposta e conquistar o território
5. Cada território conquistado = 1 estrela ⭐
6. Conquiste todos os 9 e torne-se uma **LENDA!**

### Sistema de Rank
| Estrelas | Título |
|----------|--------|
| 0-3 | 🌱 Iniciante |
| 4-6 | 🗺️ Explorador |
| 7-8 | 🏅 Mestre |
| 9 | 👑 LENDA! |

### Modo Multiplayer (Casa)
Jogue com amigos e família:
1. Cada jogador escolhe um território
2. Quem acertar a charada conquista o território
3. O jogador com mais conquistas vence!
